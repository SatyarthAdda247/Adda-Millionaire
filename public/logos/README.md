# App Logos

Please add the following logo images to this directory:

1. **reevo-logo.png** - Reevo app logo
   - Recommended size: 200x200px or higher
   - Format: PNG with transparent background preferred
   - Source: Download from Reevo's official website or app store listing

2. **adda247-logo.png** - Adda247 app logo
   - Recommended size: 200x200px or higher
   - Format: PNG with transparent background preferred
   - Source: Download from Adda247's official website (https://www.adda247.com) or app store listing

3. **learners-adda-logo.png** - Learner's Adda app logo
   - Recommended size: 200x200px or higher
   - Format: PNG with transparent background preferred
   - Source: Download from Learner's Adda official website or app store listing

## How to Add Logos

1. Download the logos from the official sources
2. Rename them to match the filenames above
3. Place them in this `/public/logos/` directory
4. The component will automatically display them

## Fallback

If a logo image is not found, the component will display the first letter of the app name as a fallback.
